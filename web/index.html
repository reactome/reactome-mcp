Available MCP Tools

This MCP server provides 53 tools for interacting with Reactome. Below is a categorized list of all available tools.

🧬 Analysis Tools

  • reactome_analyze_identifier - Analyze single gene/protein
  • reactome_analyze_identifiers - Batch enrichment analysis
  • reactome_get_analysis_result - Retrieve results by token
  • reactome_analysis_found_entities - Get matched entities
  • reactome_analysis_not_found - Get unmatched identifiers
  • reactome_analysis_resources - Resource summary
  • reactome_compare_species - Cross-species comparison
  • reactome_analysis_pathway_sizes - Size distribution
  • reactome_filter_analysis_pathways - Filter by pathway

🗺️ Pathway Tools

  • reactome_get_pathway - Get pathway details
  • reactome_top_pathways - List top-level pathways
  • reactome_pathway_ancestors - Get pathway hierarchy
  • reactome_pathway_contained_events - List sub-events
  • reactome_pathways_for_entity - Find pathways for entity
  • reactome_diagram_pathways_for_entity - Pathways with diagrams
  • reactome_events_hierarchy - Full event tree

🔍 Search Tools

  • reactome_search - Full-text search
  • reactome_search_paginated - Paginated search
  • reactome_search_suggest - Auto-complete
  • reactome_search_spellcheck - Spelling suggestions
  • reactome_search_facets - Get search facets
  • reactome_search_pathways_of - Find pathways for entity
  • reactome_search_diagram - Search within diagram

🧫 Entity Tools

  • reactome_get_entity - Get entity details
  • reactome_complex_subunits - List complex components
  • reactome_entity_other_forms - Get variant forms
  • reactome_entity_component_of - Find containers
  • reactome_participants - Event participants
  • reactome_participating_physical_entities - Physical entities
  • reactome_reference_entities - External references
  • reactome_complexes_containing - Find complexes

📤 Export Tools

  • reactome_export_diagram - Export as PNG/SVG/JPG
  • reactome_export_reaction - Export reaction diagram
  • reactome_export_fireworks - Pathway overview
  • reactome_export_sbgn - Export to SBGN
  • reactome_export_sbml - Export to SBML
  • reactome_export_pdf - Export documentation
  • reactome_export_analysis_report - PDF report
  • reactome_export_analysis_csv - CSV export
  • reactome_export_analysis_json - JSON export

🔗 Interactor Tools

  • reactome_psicquic_resources - List PSICQUIC services
  • reactome_psicquic_summary - Interaction summary
  • reactome_psicquic_details - Interaction details
  • reactome_static_interactors - Curated interactions
  • reactome_interactor_pathways - Interactor pathways
  • reactome_interactor_summary - Interaction count

⚙️ Utility Tools

  • reactome_species - List species
  • reactome_diseases - List diseases
  • reactome_database_info - Database version
  • reactome_mapping_pathways - Map to pathways
  • reactome_mapping_reactions - Map to reactions
  • reactome_orthology - Get orthologues
  • reactome_query - Generic query
